## General

  - **Login**
    - Request
      - POST
      - `/API/login`
    - Parameters
      - ID
      - password
    - Returns
      - {token}

## For users

  - **Register**
    - Request
      - POST
      - `/API/user/register`
    - Parameters
      - ID
      - password
    - Returns
      - `true` on success.
      - `false` on failure.
